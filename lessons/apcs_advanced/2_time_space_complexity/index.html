🕐 L2: 時間與空間複雜度

透過氣泡排序學習演算法複雜度分析

📖 課程簡介

時間複雜度和空間複雜度是評估演算法效率的重要指標。本課程將透過氣泡排序這個經典演算法， 讓你深入理解如何分析演算法的執行時間和記憶體使用量。

氣泡排序是一個簡單但重要的排序演算法，雖然效率不高，但非常適合用來學習複雜度分析的概念。 透過視覺化的方式，你將清楚看到每個步驟的比較和交換操作，並學會計算演算法的複雜度。

🎯 學習目標

  • 理解時間複雜度和空間複雜度的定義與重要性
  • 掌握氣泡排序演算法的工作原理和實作方法
  • 學會分析最佳、最差和平均情況下的複雜度
  • 透過計算比較次數和交換次數來理解複雜度計算
  • 培養分析其他演算法複雜度的能力

🫧 氣泡排序演算法

氣泡排序是一種簡單的排序演算法，它重複地遍歷要排序的陣列，比較相鄰的元素並在順序錯誤時交換它們。 這個過程會重複進行，直到不再需要交換為止。

複雜度分析

情況 時間複雜度 比較次數 交換次數 說明
最佳情況 O(n) n-1 0 陣列已經排序好
平均情況 O(n²) n(n-1)/2 n(n-1)/4 隨機排列的陣列
最差情況 O(n²) n(n-1)/2 n(n-1)/2 陣列反向排序

空間複雜度： O(1) - 只需要常數額外空間
