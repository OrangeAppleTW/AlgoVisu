什麼是二元樹？

二元樹（Binary Tree）是一種重要的樹狀資料結構，每個節點最多只能有兩個子節點， 分別稱為左子樹和右子樹。二元樹具有遞迴的性質，每個子樹本身也是一個二元樹。

透過視覺化演示，你將學會如何理解樹的結構，並掌握三種重要的樹走訪方法： 前序走訪（Preorder）、中序走訪（Inorder）、 以及後序走訪（Postorder），理解遞迴在樹結構中的應用。

學習樹走訪

三種樹走訪方式比較

前序走訪 (Preorder)

順序: 根 → 左子樹 → 右子樹

特點: 先處理根節點

應用: 複製樹、前綴表達式

中序走訪 (Inorder)

順序: 左子樹 → 根 → 右子樹

特點: 根節點在中間處理

應用: 二元搜尋樹排序

後序走訪 (Postorder)

順序: 左子樹 → 右子樹 → 根

特點: 最後處理根節點

應用: 刪除樹、後綴表達式
